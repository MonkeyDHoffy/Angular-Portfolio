# PageContainerComponent

Die Page-Container-Komponente sorgt für ein zentriertes Layout mit maximaler Breite und Padding. Sie nimmt beliebigen Content auf und stellt ihn im Hauptbereich dar.

## SCSS-Klasse
- `.page-container`: Hauptcontainer für Seiteninhalte

## Verwendung
```html
<app-page-container>
  <!-- Content hier -->
</app-page-container>
```
